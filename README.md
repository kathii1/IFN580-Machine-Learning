## Project description
This repository contains my assignment solutions for the module IFN580 – Machine Learning.

The project is divided into two main parts:

### Assignment 1 (Assignment1/):
Data preparation and logistic regression for a car seller scenario, where the goal is to predict whether a car purchase will be a kick (bad buy) or not. The workflow covers cleaning and preprocessing the dataset (kick.csv) and training a binary classifier to support the buy/no‑buy decision.

### Assignment 2 (Assignment2/):
A collection of five machine learning tasks implemented in separate notebooks, including a Q&A system with T5, clustering, dimensionality reduction, time series prediction with LSTM, and tweet classification with BERT. The notebook All-exercises-in-one-file.ipynb combines all exercises in a single file.
